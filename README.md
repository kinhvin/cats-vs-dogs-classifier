# Cats vs Dogs Classifier

A simple CNN image classifier for cats vs dogs using TensorFlow/Keras.

## Tech Stack
- Python
- TensorFlow / Keras
- Pillow (PIL)
- NumPy
- Matplotlib

## Files
- Notebook: [cats_vs_dogs_classifier.ipynb](cats_vs_dogs_classifier.ipynb)
- Saved model: [my_model.keras](my_model.keras) (generated)

## Dataset
Place your data under:
```
cats-vs-dogs/
  train/
    Cat/
    Dog/
```
Personally, I used a dataset I found on Kaggle: https://www.kaggle.com/datasets/bhavikjikadara/dog-and-cat-classification-dataset?select=PetImages

"Download" -> "Download dataset as zip" (or use any of the other provided options)

## Run
Open and run the notebook:
- [cats_vs_dogs_classifier.ipynb](cats_vs_dogs_classifier.ipynb)

## Output
Trains a CNN and saves the model to `my_model.keras`.
